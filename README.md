### 🚀 O que é o VortexCloud?

O **VortexCloud** é um servidor de nuvem pessoal *self-hosted* projetado para rodar localmente ou na rede de sua casa. Ele oferece uma alternativa leve, segura e rápida para armazenar e organizar arquivos diretamente no seu disco rígido, acessando tudo de qualquer celular ou computador na mesma rede Wi-Fi através do navegador.

---

### ✨ Funcionalidades Principais

- **Área de Drag & Drop (Arrastar e Soltar):** Upload prático de arquivos simplesmente arrastando-os para a tela ou selecionando-os do dispositivo.
- **Barra de Progresso em Tempo Real:** Comunicação via AJAX (`XMLHttpRequest`) que calcula e exibe em tempo real a velocidade e a porcentagem de upload de cada arquivo.
- **Galeria e Visualizador Embutidos:** Detecta arquivos de imagem, vídeo, áudio ou PDF, atribuindo ícones específicos para cada um. Possui visualizador de imagens integrado por modal.
- **Estatísticas de Disco:** Painel que exibe a contagem de arquivos e a soma total do tamanho utilizado em disco de forma dinâmica.
- **Prevenção de Conflitos:** Renomeia automaticamente arquivos duplicados adicionando um sufixo numérico para evitar substituições acidentais.

---

### 🛠️ Tecnologias Utilizadas

- **Servidor (Backend):** Node.js, Express, `multer` (tratamento de multipart/form-data)
- **Interface (Frontend):** HTML5, CSS3 (Vanilla com Glassmorphism), JavaScript ES6 (AJAX/Fetch), Lucide Icons

---

### ⚙️ Como Instalar e Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/Vortexyweb/personal-cloud.git
   cd personal-cloud
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor da nuvem:
   ```bash
   npm start
   ```

4. Acesse no seu navegador:
   👉 **`http://localhost:3001`**
